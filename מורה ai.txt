    var history = data.history || [];

                       "2. סימני מתמטיקה: לכפל השתמש אך ורק בסימן × (איקס קטן). לחילוק השתמש בסימן ÷. לעולם אל תשתמש בכוכבית (*) לכפל.\n" +

                       "4. מבנה: הסבר מילולי קצר (RTL) -> שורת רווח -> נוסחה/חישוב בתיבת LTR -> המשך קצר.\n" +
                       "5. כיווניות: בתוך [LTR] כתוב תמיד משמאל לימין בצורה אסתטית עם רווחים בין הסימנים.\n" +

        var textContent = history[i].content || "";
        
        if (i === 0 && role === "user") {
          textContent = "הנחיות מערכת: " + systemPrompt + "\n\nהודעת התלמיד:\n" + textContent;
        }
        
        var parts = [{ "text": textContent }];

        if (i === history.length - 1 && imageBase64 && imageBase64.includes(",")) {
          var base64Parts = imageBase64.split(",");
          parts.push({

              "mime_type": base64Parts[0].split(":")[1].split(";")[0],
              "data": base64Parts[1]
            }
          });
        }
        
        formattedContents.push({ "role": role, "parts": parts });
      }
    } else {
      var fallbackParts = [{ "text": systemPrompt + "\n\nהודעת התלמיד:\n" + userMessage }];

      var apiUrl = "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent?key=" + apiKey;
      
      try {
        var response = UrlFetchApp.fetch(apiUrl, options);
        var responseCode = response.getResponseCode();
        
        if (responseCode === 200) {
          var responseText = response.getContentText();
          var tempResult = JSON.parse(responseText);
          
          // מוודאים שהתשובה באמת מכילה תוכן לפני שמכריזים על הצלחה
          if (tempResult.candidates && tempResult.candidates[0] && tempResult.candidates[0].content) {
            result = tempResult;
            isSuccess = true;
            break; // הצלחה! יוצאים מהלולאה, האינדקס נשאר על המפתח המנצח
          }
        }
        
        // אם הגענו לכאן - הייתה בעיה עם המפתח (נגמרה המכסה, שגיאת שרת וכו')
        // אנחנו מקדמים את האינדקס בשקט למפתח הבא, נותנים לנוכחי לנוח, וממשיכים בלולאה
        currentIndex = (currentIndex + 1) % apiKeys.length;
        props.setProperty('API_KEY_INDEX', currentIndex.toString());
        
      } catch (innerErr) {
        // שגיאת רשת פנימית או בעיה אחרת שזרקה קריסה. 
        // מדלגים בשקט למפתח הבא ומנסים שוב.
        currentIndex = (currentIndex + 1) % apiKeys.length;
        props.setProperty('API_KEY_INDEX', currentIndex.toString());
      }
    }

    if (!isSuccess) {

        .setMimeType(ContentService.MimeType.JSON);
    }

    // עיבוד התשובה המוצלחת מה-AI
    var aiReply = result.candidates[0].content.parts[0].text;
    
    // 1. ניקוי סימני Markdown
    aiReply = aiReply.replace(/[\*\$\_]/g, "");

    aiReply = aiReply.replace(/ \* /g, " × ").replace(/\*/g, "×");

    });

    return ContentService.createTextOutput(JSON.stringify({ "reply": aiReply }))
      .setMimeType(ContentService.MimeType.JSON);
      
  } catch (err) {

  }
}
